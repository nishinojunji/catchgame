## prompt
```prompt
むしとりゲームをJavaScriptとHTMLで

ルール
2Dの画面上にプレイヤーのアミ（丸型）がある
上下左右に動かせる
画面上にランダムに虫が5匹いる（小さい丸）
制限時間は30秒で29, 28 と減っていき0でゲームオーバー
アミを移動して、虫に被せ、「キャッチ」ボタンを押すとキャッチする。
１点追加され、虫は消える
１匹取ると新しい虫が１匹出現する。

ランダムに四角い障害物が2つある。
プレイヤーは障害物に入れない。
虫は障害物のところに発生しない。
１匹取ると制限時間が１秒増える。
```
